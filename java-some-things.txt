Collision is a phenomenon where more than 1 element fits the same bucket, that array becomes the head of the LinkedList and the second element is added there, more number of elements in same bucket then it will become a tree in order to reduce the time required for fetching.

***Annotations


Java 5 introduced annotations, it is a way to provide metadata/maintaining information about the code to the developers, to compiler, to JVM or spring.

Eg, conside a simple @Override annotation, it tells us that we need to override the following method, the compiler also flags if we make any error, we can add annotation only to the specific places.

**How to create an annotation, its simple create an interface and add @ at the start of interface
eg:

@interface OnePlus{
	
}

Need to specify 2 things, where to apply the annotation and its reach.

**Meta annotations are annotations used to create annotations, 
* @Target() is a meta annotation which specifies where this annotation is applicable, it expects an enum of ElementType (field, constructor, class)
* @Retention() is retention policy, till when you want the annotations power, it expects RetentionPolicy enum (runtime, compile time)

Sometimes we need to pass values to the annotation, this are called one valued, 2 valued or multi valued annotations, we provide the attribute in the annotation

syntax:

datatype var_name();
or 
datatype var_name() default value;

eg: String name();
String name() default "Harsh";

-We need to provide this else we can't use the annotation, this doesn't apply if they have default values.

* we can provide the multiple enums in curly braces, for the meta annotations. 
eg; @Target({ElementType.TYPE,ElementType.CONSTRUCTOR})
